# Projeto Ateliê de Cartas Felpojam

## Nome do jogo e Equipe

Jogo: **Ateliê de Cartas** (Letter Attelier)
Equipe: **astroFrog**

## Descrição e Instruções

Ateliê de cartas é um jogo sobre expressar sentimentos através de palavras, como a forma que uma mensagem é transmitida muda o sentimento de quem lê.

Todo o jogo é transmitido através de cartas, seus clientes fazem os pedidos e indicam qual é a mensagem que eles gostariam de passar para seus destinatários.  

Então você recebe a carta, a lê, e a cena de começar a escrever começa, é um mini game de escolha de frases, você escolhe qual frase se encaixa melhor no contexto. E claro, não poderíamos nos esquecer dos carimbos! Você tem opções de carimbos para decorar toda a carta com eles.

Após terminar você envelopa a carta, você pode carimbar o envelope também. Mas para finalizar a carta você precisa do carimbo de sinete!

Para usá-lo é só escolher a cor da cera, levá-la ao fogão, após derretida a coloque no envelope, e escolha qual carimbo você gostaria de utilizar.  

Carimbada e enviada, escolha a próxima carta! Mas cuidado, escolher opções não condizentes com o sentimento do cliente no mini game das cartas pode fazer você perder alguns itens colecionáveis!
